# Cone Volume Calculator

A simple, lightweight C program that calculates the volume of a cone based on user-defined radius and height inputs.

## Description

This program prompts the user to enter the radius and height of a cone, processes the inputs using the standard geometric formula, and outputs the resulting volume rounded to two decimal places.

The calculation uses the standard formula:
$$V = \frac{1}{3} \pi r^2 h$$

Where:
*   $V$ = Volume
*   $r$ = Radius
*   $h$ = Height

## Prerequisites

To compile and run this program, you need a C compiler (such as `gcc`) installed on your system. 

> **Note:** This program utilizes the `<math.h>` library for the `M_PI` constant. Depending on your compiler, you may need to explicitly link the math library during compilation.
